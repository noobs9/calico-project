# calico-project

## description
練習用Web app。
todo的なものをみんなでPOSTできる。

- RDBとやりとりしてCRUDできる
- JWTで認証できる
- 極力画面遷移なし
- パッと見質素
- 動きがヌルヌル
- サクサク動いてノンストレス
- https対応(オレオレ証明書)

完成品のイメージはこんな感じ。
最初なので深堀はなるべくしないでガンガン進める。
